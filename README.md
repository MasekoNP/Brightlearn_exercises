# Brightlearn_exercises
This repository showcases the exercises I completed during the BrightLearn course, highlighting my learning and practical application of key concepts. The assignments are handwritten, illustrating my detailed thought process and foundational understanding.
Exercise 1 focused on SQL fundamentals, specifically retrieving data using SELECT statements and applying filtering conditions to refine query results.
Exercise 2 focused on SQL fundamentals, with an emphasis on using aggregate functions (such as MIN, MAX, SUM, AVG and COUNT) and GROUP BY statements to organize and derive insights from data.
Exercise 3 focused on SQL fundamentals, with an emphasis on using CASE statements to apply conditional logic and categorize data within queries.
Exercise 4 focused on SQL fundamentals, with an emphasis on using joins to combine tables, UNION to merge query results, filtering to refine data and aggregate functions to summarize insights.
